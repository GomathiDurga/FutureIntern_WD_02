My Projects

Waste Segregation using ARDUINO

This project aims to automate the segregation of waste into wet and dry categories using an Arduino microcontroller and a soil moisture sensor. The system operates by measuring the moisture level in waste materials. When wet waste is detected (indicated by a high moisture reading), the Arduino activates a motorized mechanism to direct it to a designated bin for wet waste. Conversely, dry waste, which shows a low moisture level, is directed to a separate bin. This efficient waste segregation system helps in promoting recycling and composting while reducing landfill waste, contributing to a cleaner environment. The project demonstrates the integration of sensors and microcontrollers for smart waste management solutions.

Certificate Generation using MATLAB

This project focuses on creating a dynamic certificate generation system using MATLAB. The application allows users to input participant details, such as name, course title, and completion date. Utilizing MATLAB's graphic capabilities, the system generates visually appealing certificates that can be customized with logos and specific designs.This certificate generation system simplifies the process of issuing certificates, making it ideal for educational institutions, workshops, and training programs. It showcases MATLAB's potential for practical applications in document automation and design.

Text to Voice Conversion Using HTML, CSS, and JavaScript

This project implements a text-to-voice conversion tool using web technologies: HTML, CSS, and JavaScript.
